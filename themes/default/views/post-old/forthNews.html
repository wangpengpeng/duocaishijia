    jquery ajax回调函数中调用$(this)的问题

    今天在用jquery的ajax的回调函数时出现了问题，就是在回调函数中不能使用$(this)。这个问题一直弄了很久，怎么也解决不了，后来是在调用函数前先把$(this)存入变量，然后再在回调函数中使用就没问题了，在这作一下笔记。

    $(document).ready(function(){
    $(“.del”).click(function () {
        obj=$(this).parent();//回调函数前先写入变量;
        delid=$(this).attr(‘delid’);
        $.get(‘drag.php’,{‘op’ : ‘del’,'id’ : delid},function(data){
            if(data==1){
                alert(‘删除成功’);
                obj.remove();//直接写$(this).parent()不会起作用。
            }else{
                alert(‘删除失败’);
            }
        });
    });
});
